# Text Adventure UI Thing

Allows you to construct text adventures in an intuitive, UI flowchart, and export to wikitext.

## What You'll Need

To get the wikitext to actually work, you must be in a MediaWiki wiki of some sort. It also needs the following:

- [Extension:DPLForum](https://www.mediawiki.org/wiki/Extension:DPLforum)
- `Template:Text adventure` — see [templates/Template_Text_adventure.wikitext](templates/Template_Text_adventure.wikitext) and [templates/Template_Text_adventure_styles.css](templates/Template_Text_adventure_styles.css)
- `Template:Text adventure choice` — see [templates/Template_Text_adventure_choice.wikitext](templates/Template_Text_adventure_choice.wikitext) and [templates/Template_Text_adventure_choice_styles.css](templates/Template_Text_adventure_choice_styles.css)
- `Template:Get` — see [templates/Template_Get.wikitext](templates/Template_Get.wikitext)

Copy the contents of each file to the corresponding template page on your wiki.
